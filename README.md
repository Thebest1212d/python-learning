# 📖 Development Workflow

For every project I follow a structured workflow:

1. Create a dedicated feature branch
2. Implement one feature at a time
3. Write meaningful commits
4. Open a Pull Request
5. Review and merge into the `main` branch

This workflow helps me practice professional version control while improving my programming skills.

# 📚 Learning Philosophy

This repository is not intended to contain perfect code.
