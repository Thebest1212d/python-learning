# Inventory Analyzer 📊

A small Python project for learning **Pandas** by analyzing computer inventory data stored in CSV files.

The project uses **dummy inventory data** and focuses on reading, filtering, and analyzing structured datasets.

---

## 🎯 Learning Goals

- Learn how to use Pandas
- Read and analyze CSV files
- Work with DataFrames
- Filter and sort data
- Calculate simple statistics
- Practice data analysis using Python

---

## 📂 Project Structure

```text
inventory-analyzer/
│
├── data/
│   └── sample_inventory.csv
│
├── main.py
└── README.md
```

---

## 🚀 Planned Features

- Load inventory data from a CSV file
- Display basic dataset information
- Show the number of devices
- Display column names
- Filter devices by category
- Search for a specific employee or device
- Generate simple statistics using Pandas

---

## 🛠 Technologies

Current:

- Python 3
- Pandas
- CSV

---

## 📖 Learning Focus

The primary goal of this project is to learn how to work with **Pandas**.

Rather than building a complete inventory management system, the focus is on understanding how to load, inspect, filter, and analyze tabular data.
